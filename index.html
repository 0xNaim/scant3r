What is Scant3r?

this is a module-based web automation tool that I made for saving my scripting time by providing some utilities that every web pentester needs in his automation script instead of focusing on ( logger, parsers, output function, cmd args, multi-threading), just write the logic of your scanning idea with scant3r utils without caring about these things

Configurable

You have the flexibility to change the options of scant3r and add your own options to it.

Cool utilities

You can use the cool utils of scant3r to make your script more efficient.
